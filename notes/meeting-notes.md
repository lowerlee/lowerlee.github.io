# Meeting Notes

A place to record key points and action items from meetings.

---

## Template

Use this format for each meeting:

```
## YYYY-MM-DD — Meeting Title

**Attendees:** Name 1, Name 2

**Agenda:**
- Topic 1
- Topic 2

**Notes:**
- Key point discussed

**Action Items:**
- [ ] Task — Owner — Due date
```
