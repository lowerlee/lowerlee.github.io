# Getting Started with My Notes Site

Today I set up this notes site using GitHub Pages and Jekyll. Here's how it works:

## How to Add Notes

1. Create a new Markdown file in the `notes/` folder (e.g. `notes/2026-03-19.md`)
2. Add a front matter block at the top:
   ```yaml
      ---
         layout: page
            title: "Your Note Title"
               date: YYYY-MM-DD
                  ---
                     ```
                     3. Write your note in Markdown below the front matter
                     4. Commit the file — GitHub Pages will automatically rebuild the site

                     ## Tips

                     - Link to notes from `index.md` to make them discoverable
                     - Use headers (`#`, `##`, `###`) to organize content
                     - Code blocks use triple backticks: ` ``` `
